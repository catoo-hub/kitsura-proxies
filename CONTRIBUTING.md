# Руководство по участию (Contributing Guide)

Спасибо за ваш интерес к проекту! Мы рады любой помощи.

## Как предложить изменения

1. **Форкните репозиторий**: Нажмите кнопку "Fork" в правом верхнем углу страницы репозитория.
2. **Клонируйте форк**: `git clone https://github.com/ВАШ_НИК/kitsura-vpn-bot.git`
3. **Создайте ветку**: `git checkout -b feature/my-cool-feature`
4. **Внесите изменения**: Напишите код, исправьте баги, улучшите документацию.
5. **Закоммитьте изменения**: `git commit -m "Добавлена крутая фича"`
6. **Сделайте пуш**: `git push origin feature/my-cool-feature`
7. **Создайте Pull Request**: Перейдите на страницу оригинального репозитория и нажмите "New Pull Request".

## Стиль кода

* Мы используем стандартный стиль PEP 8 для Python.
* Пожалуйста, добавляйте комментарии к сложным участкам кода.

## Сообщение об ошибках

Если вы нашли ошибку, пожалуйста, создайте Issue с подробным описанием проблемы, шагами по воспроизведению и логами (если есть).

Спасибо за ваш вклад! ❤️
